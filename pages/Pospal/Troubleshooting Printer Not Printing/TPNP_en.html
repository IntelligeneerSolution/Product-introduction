Printer Troubleshooting Steps

Nov 13, 2024 · Charlie




  1. Check if the store's network is functioning properly
  2. Ensure the printer is turned on
  3. Check if the printer's network cable is connected
  4. Verify that the printer's IP address matches the IP address in the Pospal system
  5. Check if the system has received the order
  6. Cross-check to determine if the issue is with the printer
  7. Out of paper
  8. If all the above steps are confirmed to be normal, then it is likely an issue with the Pospal system
